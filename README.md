# clinica-odontologica-app-android

# Aplicativo Android – Clínica Odontológica

Este repositório contém o código-fonte do aplicativo Android desenvolvido
no contexto de um projeto acadêmico de extensão universitária.

## Objetivo
Digitalizar processos administrativos e de atendimento de uma clínica odontológica,
permitindo melhor organização de pacientes, agenda e informações clínicas.

## Tecnologias Utilizadas
- Kotlin
- Android Studio
- Firebase (simulado)
- Arquitetura MVVM
- Test-Driven Development (TDD)

## Funcionalidades
- Cadastro de pacientes
- Listagem de pacientes
- Estrutura base para controle de agenda
- Testes unitários

## Contexto Acadêmico
Projeto desenvolvido como parte das atividades extensionistas,
integrando teoria e prática em desenvolvimento de software.
